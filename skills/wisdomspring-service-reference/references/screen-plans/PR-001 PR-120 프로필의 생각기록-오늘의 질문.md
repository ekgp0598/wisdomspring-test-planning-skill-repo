# PR-001 / PR-120 프로필의 생각기록-오늘의 질문

| 항목 | 내용 |
| --- | --- |

| 연관 화면 | M-FD-010, W-FD-010, M-FD-B030, W-FD-M030, M-PR-P110, W-PR-P110, M-PR-420, W-PR-420 |

## 화면 개요

프로필 화면 내 `오늘의 질문` 기록 영역으로, 사용자가 작성한 오늘의 질문 답변 이력을 월별로 모아 확인할 수 있는 화면이다. 프로필 정보, 멤버십 상태, 오늘의 질문 답변 작성/수정 진입, 탭별 집계 정보, 질문-답변 리스트를 함께 제공한다.

## 상세 설명

### 1. 프로필 정보

- 상단에 아래 프로필 정보를 노출한다.
  - 프로필 사진
  - 프로필 수정 버튼
  - 닉네임
- `수정` 클릭 시 프로필 수정 팝업을 호출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 모바일 | M-PR-P110 |
| 웹 | W-PR-P110 |

### 2. 구독 정보

#### 구독 중이 아닐 때

- `현재 멤버십이 아닙니다.` 문구를 노출한다.
- 클릭 시 멤버십 가입 페이지로 이동한다.

| 디바이스 | 이동 화면 |
| --- | --- |
| 모바일 | M-PR-420 |
| 웹 | W-PR-420 |

#### 구독 중일 때

- 원형 아이콘 ON 상태를 노출한다.
- `구독중` 텍스트를 노출한다.
- 멤버십 기간을 `YYYY.MM.DD~YYYY.MM.DD` 형식으로 노출한다.
- 예시: `● 구독중 2026.01.01~2026.03.10`

### 3. 오늘의 질문 답변 작성 버튼

- 답변을 하지 않았을 때 버튼 문구는 `오늘의 생각 쓰기`이다.
- 답변을 했을 때 버튼 문구는 `오늘의 생각 수정하기`이다.

### 3-1. 타 사용자 프로필 조회 상태

#### 타 사용자 프로필 조회 시 - 로그인(회원)

- 구독 정보 대신 팔로잉 여부 정보를 노출한다.
- 프로필 수정 아이콘은 노출하지 않는다.
- `오늘의 생각 수정하기` 버튼은 노출하지 않는다.

#### 타 사용자 프로필 조회 시 - 비로그인(비회원)

- 팔로잉 여부 정보도 노출하지 않는다.
- 프로필 수정 아이콘은 노출하지 않는다.
- `오늘의 생각 수정하기` 버튼은 노출하지 않는다.

### 4. 탭 명칭 변경 및 집계 기준

- `생각기록` 탭은 `기록`으로 명칭 변경한다.
- `기록` 탭에는 `답변 + 노트 + 댓글` 개수의 합계를 표시한다.
- `오늘의 질문` 탭 명칭은 유지한다.
- `오늘의 질문` 탭에는 사용자가 남긴 답변 총 개수를 표시한다.
- `발견노트` 탭은 `노트`로 명칭 변경한다.
- `노트` 탭에는 사용자가 남긴 노트 총 개수를 표시한다.
- `댓글` 탭 명칭은 유지한다.
- `댓글` 탭에는 사용자가 남긴 댓글 총 개수를 표시한다.
- `관심콘텐츠` 탭 명칭은 유지한다.
- `관심콘텐츠` 탭에는 북마크한 콘텐츠 개수를 표시한다.
- `팔로우` 탭은 `팔로워`로 명칭 변경한다.
- `팔로워` 탭에는 사용자를 팔로우하는 사람 수를 표시한다.

### 5. 질문-답변 리스트

- 질문과 답변은 월별로 그룹핑하여 노출한다.
- 각 월별로 해당 월에 몇 개의 답변을 작성했는지 표시한다.
- 월 그룹 표기 방식은 `YYYY-MM 질문`이다.
- 질문 기준 날짜는 답변 작성일이 아니라 질문 날짜를 사용한다.

#### 날짜 표기

- 각 질문의 날짜는 `일자(숫자 2자리) + 요일(영문 3자 대문자 약어)` 형식으로 표시한다.
- 예시: `11 SUN`

#### 질문-답변 영역

- 질문 영역과 답변 영역은 구분하여 노출한다.
- 클릭 활성화는 답변 영역에만 적용한다.
- 답변 클릭 시 답변 조회 화면을 호출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 모바일 | M-FD-010 |
| 웹 | W-FD-010 |

- 답변은 최대 2줄까지 노출한다.
- 2줄을 초과하면 말줄임 처리한다.

### 6. 답변 더보기 메뉴

- 각 답변에는 더보기 메뉴를 제공한다.
- 더보기 메뉴에서 `수정`, `삭제` 기능을 제공한다.

#### 수정

- 수정 선택 시 답변 작성 화면을 호출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 모바일 | M-FD-B030 |
| 웹 | W-FD-M030 |

#### 삭제

- 삭제 선택 시 확인 알럿을 노출한다.
- 알럿 문구는 `삭제하시겠습니까?`
- 액션은 `취소`, `확인` 두 가지를 제공한다.

## 비고

- `PR-001`과 `PR-120`의 역할 구분이 명확하지 않아 공통 문서로 정리했다. 필요 시 메인 프로필과 탭 상세 화면으로 분리 정의할 수 있다.
- 월별 그룹 헤더에 답변 개수 표기 형식이 구체적으로 정해지지 않아 추가 정의가 필요하다.
