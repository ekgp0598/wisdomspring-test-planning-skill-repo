# M-IN-001 인트로 첫 화면

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 화면 ID | M-IN-001 |
| 화면명 | 인트로 첫 화면 |
| 화면 유형 | 페이지 |
| 지원 디바이스 | 모바일 전용 |

## 화면 개요

스플래시 이후 노출되는 모바일 전용 인트로 화면이다. 오늘의 질문과 위즈덤스프링 답변, 실사용자 최신 답변을 함께 노출하며, 사용자는 답변 작성 또는 홈 진입을 선택할 수 있다.

## 상세 설명

### 1. 오늘의 질문

- 스플래시 이후 `오늘의 질문`을 노출한다.
- 질문 텍스트는 Admin에서 관리한다.
- 질문 글자 수는 Admin에서 최대 50자로 제한한다.
- 서비스 화면에서는 질문을 최대 3줄까지 노출한다.
- 3줄을 초과하면 텍스트를 잘라서 노출한다.

### 2. 위즈덤스프링 답변

- 오늘의 질문에 대한 위즈덤스프링 답변을 메인에 노출한다.
- 답변 값은 Admin에서 설정한다.
- 답변 본문은 최대 5줄까지 노출한다.
- 5줄을 초과하면 말줄임 처리한다.

### 3. 실사용자 최신 답변 슬라이드

- 위즈덤스프링 답변 좌우로 실사용자의 최신 답변을 슬라이드 형태로 노출한다.
- 각 답변 카드의 구성 요소는 아래와 같다.
  - 사용자 프로필
  - 닉네임
  - 답변 내용
- 답변 내용은 최대 5줄까지 노출하며, 초과 시 말줄임 처리한다.
- 실사용자 답변이 없으면 해당 영역은 노출하지 않는다.
- 카드 배열 규칙은 아래와 같다.
  - 위즈덤스프링 답변 카드의 오른쪽부터 최신순으로 배치한다.
  - 실사용자 답변은 최대 9개까지 노출한다.
  - 전체 카드 수는 최대 10개이다.
  - 위즈덤스프링 답변 카드의 왼쪽에는 9번째 카드를 배치한다.

### 4. 버튼 및 이동

#### 내 생각 쓰기

- `내 생각 쓰기` 선택 시 오늘의 질문 바텀시트를 호출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 모바일 | M-FD-B030 |

- 저장 완료 후 질문피드로 이동한다.

#### 콘텐츠 둘러보기

- `콘텐츠 둘러보기` 선택 시 홈으로 이동한다.
- 앱을 재접속하면 인트로 화면이 다시 노출된다.

#### 오늘 하루 보지 않기

- `오늘 하루 보지 않기` 선택 시 홈으로 이동한다.
- 앱을 재접속해도 당일에는 인트로 화면이 노출되지 않는다.

## 비고

- `오늘 하루 보지 않기`의 유지 기준이 앱 재실행인지, 날짜 기준 자정 초기화인지 추가 확인이 필요하다.
- 실사용자 최신 답변 슬라이드의 정확한 순환 방식과 초기 포커스 위치는 별도 정의가 필요하다.
