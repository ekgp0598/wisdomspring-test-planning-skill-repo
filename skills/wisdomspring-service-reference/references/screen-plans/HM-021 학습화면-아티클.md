# HM-021 학습화면-아티클

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면 ID | HM-021 |
| 모바일 화면 ID | M-HM-021 |
| 웹 화면 ID | W-HM-021 |
| 화면명 | 학습화면-아티클 |
| 화면 유형 | 페이지 |

## 화면 개요

사용자가 아티클 콘텐츠를 클릭하여 진입했을 때 노출되는 학습화면이다. 기본 구조는 `HM-020 학습화면-영상`과 동일하되, 상단 콘텐츠 정보 영역과 본문 영역이 아티클 형식에 맞게 구성된다.

## 상세 설명

### 1. 상단 콘텐츠 정보

- 아티클 콘텐츠는 Admin에서 HTML 형식으로 편집한다.
- 영상 콘텐츠 학습화면과 비교했을 때 상단 영역 구성이 다르다.
- 상단에는 아래 항목을 노출한다.
  - 테마명
  - 전체보기
  - 서브테마명
  - 콘텐츠 제목
  - 저자 프로필
  - 저자명
  - 직함
  - 분량 분수
  - 요약글
  - 썸네일
- 분량 분수는 Admin에서 설정한다.
- 요약글 글자 수 제한은 Admin에서 관리한다.

### 2. 본문

- 본문은 Admin에서 HTML로 편집한 데이터를 노출한다.
- 본문 하단에는 태그를 노출한다.

### 3. 공통 영역

- 상단 영역과 본문을 제외한 이하 영역은 `HM-020 학습화면-영상`과 동일하게 구성한다.
- 공통 적용 범위는 아래와 같다.
  - 이전/다음 화 보기
  - 댓글 영역
  - 추천 콘텐츠
  - 기타 하단 인터랙션 영역

## 비고

- `전체보기`의 동작 방식과 연결 화면은 추가 정의가 필요하다.
- 아티클 화면에서 `HM-020`의 플레이어 영역을 어떤 UI로 대체하는지 별도 확인이 필요하다.
