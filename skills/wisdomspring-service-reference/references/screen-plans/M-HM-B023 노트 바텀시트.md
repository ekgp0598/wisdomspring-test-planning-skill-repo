# M-HM-B023 노트 바텀시트

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 화면 ID | M-HM-B023 |
| 화면명 | 노트 바텀시트 |
| 화면 유형 | 바텀시트 |

## 화면 개요

학습화면-영상에서 노트 영역 클릭 시 플레이어 하단에 펼쳐지는 모바일 전용 바텀시트이다. 바텀시트가 열려 있는 동안에도 동영상은 계속 재생되며, 노트 리스트 확인, 좋아요, 노트 작성, 노트 댓글 확인 기능을 제공한다.

## 상세 설명

### 1. 기본 동작

- 노트(100개) 영역 클릭 시 플레이어 하단 높이의 1단계 바텀시트를 노출한다.
- 1단계 오픈 시 백그라운드는 딤 처리하지 않는다.
- 바텀시트가 열린 상태에서도 동영상은 계속 재생된다.
- 바텀시트를 슬라이드 업하면 2단계 높이로 확장된다.
- 2단계 상태에서도 동영상은 계속 재생된다.

### 2. 노트 리스트

- 노트 리스트를 노출한다.
- 각 노트는 본문 미리보기를 최대 4줄까지 노출한다.
- 4줄을 초과하면 말줄임 처리한다.
- `나`의 노트는 언제나 최상단에 고정한다.
- `나`의 노트는 회색 배경으로 구분한다.

### 3. 노트 입력

- 노트 입력 필드 클릭 시 현재 바텀시트 높이는 유지한다.
- 입력 필드와 키패드를 함께 펼친다.
- 입력 상태에서는 백그라운드를 딤 처리한다.
- 입력 중에도 동영상은 계속 재생된다.
- 줄바꿈은 허용하지 않는다.
- 입력창은 최대 3줄까지 작성 가능하며, 이후에는 스크롤 처리한다.
- 글자 수는 최대 500자로 제한한다.
- `리뷰 쓰기` 버튼을 제공한다.

### 4. 인터랙션

- 좋아요 아이콘 클릭 시 카운트가 증가한다.
- 좋아요 아이콘은 filled 상태로 변경한다.
- 노트의 댓글 아이콘 클릭 시 해당 노트 댓글 페이지(댓글 리스트)로 진입한다.
- 사용자 경험은 노트 리스트 우측으로 선택한 노트 상세 화면에 들어가는 것처럼 구성한다.
- 노트 상세 페이지에서는 노트 리스트로 뒤로가기 할 수 있어야 한다.
- `노트 댓글 전체 펼침` 기능을 제공한다.

### 5. 빈 상태

- 등록된 노트가 없을 경우 아래 메시지를 노출한다.
- `등록된 노트가 없습니다. 첫 번째 노트를 남겨보세요.`

## 비고

- 노트 상세 페이지의 별도 화면 ID가 있는지 추가 확인이 필요하다.
