# HM-010 테마 상세 페이지

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면 ID | HM-010 |
| 모바일 화면 ID | M-HM-010 |
| 웹 화면 ID | W-HM-010 |
| 화면명 | 테마 상세 페이지 |
| 화면 유형 | 페이지 |

## 화면 개요

선택한 테마의 소개 정보, 현재 시청 중인 콘텐츠 진입 기능, 서브테마 및 서브테마별 콘텐츠 리스트를 제공하는 화면이다. 사용자는 이 화면에서 테마의 전체 구성과 학습 진행 상태를 확인하고, 원하는 서브테마 또는 콘텐츠로 진입할 수 있다.

## 상세 설명

### 1. 테마 소개

- 테마 기본 정보를 상단에 노출한다.
- 구성 항목은 아래와 같다.
  - 카테고리
  - 테마명
  - 테마 썸네일
  - 테마 설명
- 테마명은 최대 10자 이내로 운영한다.
- 테마 설명은 최대 100자 이내로 운영한다.
- 테마 설명 노출은 PC, 모바일 모두 3줄 고정으로 처리한다.
- 3줄을 초과하는 경우 말줄임이 아니라 잘려서 보이지 않도록 처리한다.

### 2. 시청 중 콘텐츠 바로가기

- 현재 시청 중인 콘텐츠를 바로 재생할 수 있는 기능을 제공한다.
- 기준 콘텐츠는 가장 최신 시청 콘텐츠이다.

#### 상태별 노출 규칙

- 시청 중 콘텐츠 정보가 있을 때
  - `시청 중인 콘텐츠` 상태로 표시한다.
  - 프로그레스바를 노출한다.
  - 남은 시간을 노출한다.
  - `이어보기` 재생 버튼을 제공한다.
- 테마 시청 전일 때
  - 테마의 첫 화 콘텐츠를 표시한다.
  - `처음부터 보기` 재생 버튼을 제공한다.
- 예외 상태
  - 2화 시청 완료 후 3화 자동재생 전 종료된 경우, 3화를 기준 콘텐츠로 표시한다.
  - 이 경우 프로그레스바와 남은 시간은 초기 상태로 표시한다.

#### 구성 항목

- 서브테마 유형 + 콘텐츠명
- 프로그레스바
- 남은 시간
- 재생 버튼
- 콘텐츠 완료 현황

#### 버튼 및 진행 상태 정의

- `처음부터 보기`
  - 테마 시청 전 상태일 때 노출한다.
  - 테마의 첫 화부터 재생한다.
- `이어보기`
  - 콘텐츠 시청 이력이 있을 때 노출한다.
  - 마지막 시청 지점부터 이어서 재생한다.
- `콘텐츠 완료`
  - 테마 내 전체 콘텐츠 대비 현재 시청 콘텐츠 개수를 표시한다.
  - 표기 형식: `시청 콘텐츠 개수 / 총 콘텐츠 개수`

### 3. 콘텐츠(서브테마 리스트)

- 테마에 속한 서브테마 리스트를 노출한다.
- 서브테마 유형 예시는 `TALK`, `BOOK`, `ESSAY` 등이다.

#### 웹 노출 항목

- 서브테마 유형
- 서브테마 제목
- 썸네일
- 서브테마 소개글
- 서브테마 소개글은 최대 100자 기준이다.
- `TALK 소개` 등 서브테마 유형 소개는 툴팁으로 제공한다.
- 툴팁은 마우스 오버 시 노출한다.

#### 모바일 노출 항목

- 서브테마 유형
- 서브테마 제목
- 썸네일
- 모바일에서는 `TALK 소개` 툴팁과 서브테마 소개글을 기본 접힘 상태에서 숨긴다.
- 해당 정보는 아코디언을 펼쳤을 때 노출한다.

#### 상태 뱃지

- 신규 오픈 서브테마
  - 오픈일부터 1주일 동안 `NEW` 뱃지를 표시한다.
- 오픈일 이전 서브테마
  - 그레이 배경의 `COMING SOON` 뱃지를 표시한다.
  - 썸네일에는 `COMING SOON` 텍스트를 함께 노출한다.
  - 썸네일 전체는 딤 처리한다.

### 4. 서브테마 내 콘텐츠 리스트

- 서브테마 클릭 시 하단에 아코디언 형태로 콘텐츠 리스트를 펼친다.

#### 영상 콘텐츠

- 썸네일
- 재생 시간
- 콘텐츠 제목
- 강사명
- 썸네일 하단에 시청 기록 프로그레스바를 바로 노출한다.

#### 아티클 콘텐츠

- 썸네일
- 콘텐츠 제목
- 요약문
- 강사명
- 분량 분수
- 요약문은 1줄만 노출하며, 초과 시 말줄임 처리한다.

### 5. 서브테마 유형 소개 툴팁

- 웹에서는 호버 시 툴팁을 노출한다.
- 모바일에서는 클릭 시 툴팁을 노출한다.

#### 대담(TALK)

전문가와 사회자가 함께하는 대담 형식 영상이다. 각 100분 분량으로 구성되며, 인문학 테마의 시작과 확산, 그 배경과 주체, 그리고 오늘날 우리 삶에까지 이어지는 영향력을 입체적이고 흥미롭게 조명한다.

#### 북러닝(BOOK)

쉬운 해설을 통해 더욱 깊게 탐독하는 인문학 필독서 해설 강의이다. 각 50분 분량으로 구성되며, 테마별로 엄선한 두 권의 고전을 영상으로 탐독하면서 단순 읽기를 넘어 고전이 인류의 사유 방식에 미친 영향력을 배울 수 있다.

#### 에세이(ESSAY)

깊이 있는 인문학 지식을 사유할 수 있는 고품질 에세이이다. 각 원고 20매 분량으로 구성되며, 인문학계의 지식을 심층 분석한 밀도 있는 글을 통해 더 넓은 관점으로 세상을 바라볼 수 있는 기회를 제공한다.

## 비고

- 테마명 10자 이내, 테마 설명 100자 이내 제한이 프론트 표시 기준인지 Admin 입력 제한인지 추가 확인이 필요하다.
- `시청 중 콘텐츠 바로가기`에서 웹/모바일 간 UI 차이 여부는 별도 정의가 필요하다.
