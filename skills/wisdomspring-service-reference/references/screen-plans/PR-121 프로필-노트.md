## 화면 개요

프로필 화면에서 사용자가 작성한 노트를 모아 확인하는 화면이다. 디자인에는 멤버십 구독 중 상태도 함께 포함되어 있으며, 노트 본문, 작성일, 수정/삭제, 연관 콘텐츠 진입 기능을 제공한다.

## 상세 설명

### 1. 구독 정보

#### 구독 중이 아닐 때

- `현재 멤버십이 아닙니다.` 문구를 노출한다.
- 클릭 시 멤버십 가입 페이지로 이동한다.

| 디바이스 | 이동 화면 |
| --- | --- |
| 모바일 | M-PR-420 |
| 웹 | W-PR-420 |

#### 구독 중일 때

- 원형 아이콘 ON 상태를 노출한다.
- `구독중` 텍스트를 노출한다.
- 멤버십 기간을 `YYYY.MM.DD~YYYY.MM.DD` 형식으로 노출한다.
- 예시: `● 구독중 2026.01.01~2026.03.10`

### 2. 노트 리스트

- 사용자가 작성한 노트 목록을 노출한다.
- 노트 본문은 최대 2줄까지 표시한다.
- 2줄을 초과하면 말줄임 처리한다.
- 노트를 남긴 날짜는 `YYYY.MM.DD` 형식으로 표시한다.

### 3. 노트 상세 조회

- 노트 클릭 시 노트 댓글 조회 화면을 호출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 모바일 | M-HM-B023 |
| 웹 | W-HM-P024 |

### 4. 더보기 메뉴

- 각 노트에는 더보기 메뉴를 제공한다.

#### 수정

- 수정 선택 시 노트 댓글 조회 팝업/바텀시트를 `EDITING` 상태로 노출한다.

#### 삭제

- 삭제 선택 시 확인 팝업을 노출한다.
- 팝업 문구는 `삭제하시겠습니까?`이다.
- 액션은 `취소`, `확인`을 제공한다.
- `확인` 선택 시 해당 노트를 리스트에서 삭제한다.

### 5. 콘텐츠 정보

- 각 노트에는 연관 콘텐츠 정보를 함께 노출한다.
- 구성 항목은 아래와 같다.
  - 썸네일
  - 제목
  - 이동 버튼
- 콘텐츠 정보 클릭 시 해당 콘텐츠 학습화면으로 이동한다.

## 비고
