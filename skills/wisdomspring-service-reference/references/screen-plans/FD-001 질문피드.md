# FD-001 질문피드

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면 ID | FD-001 |
| 모바일 화면 ID | M-FD-001 |
| 웹 화면 ID | W-FD-001 |
| 화면명 | 질문피드 |
| 화면 유형 | 페이지 |

## 화면 개요

매일 제시되는 사유의 질문에 대해 사용자가 답변을 남기고 서로 소통할 수 있는 질문 피드 페이지이다. 오늘의 질문 영역, 최근 질문 리스트, 데일리 답변 리스트, 연관 오늘의 콘텐츠로 구성되며 페이지는 무한 스크롤 방식으로 동작한다.

## 상세 설명

### 1. 기본 동작

- 페이지는 무한 스크롤 방식으로 동작한다.

### 2. 오늘의 질문 영역

- `Daily 오늘의 질문`을 노출한다.
- 타이틀은 `오늘의 질문`으로 고정한다.
- 날짜는 기본 상태에서는 노출하지 않는다.
- 질문 텍스트 데이터는 Admin에서 관리한다.
- 질문 텍스트는 Admin에서 최대 50자 제한 예정이며, 영역을 초과할 경우 프론트에서 말줄임표 처리한다.
- `내 생각 쓰기` 버튼을 제공한다.

#### 버튼 액션

- `내 생각 쓰기` 버튼 클릭 시 입력 화면을 호출한다.

| 디바이스 | 호출 화면 | 방식 |
| --- | --- | --- |
| 모바일 | M-FD-B030 | 바텀시트 |
| 웹 | W-FD-M030 | 모달 오버레이 |

- 작성 완료 후 버튼 문구는 `내 생각 쓰기`에서 `내 답변 보기`로 변경한다.
- `내 답변 보기` 클릭 시 내 답변 상세 페이지를 호출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 모바일 | M-FD-010 |
| 웹 | W-FD-010 |

#### 질문 플로팅/상단 고정

- 페이지 스크롤 시 오늘의 질문 카드는 축소되며 상단 고정된다.
- 축소된 상태에서는 아래 요소를 노출한다.
  - `오늘의 질문 / 0000.00.00 질문`
  - 질문 텍스트
  - `내 생각 쓰기` 또는 `내 답변 보기` 버튼
- 축소 상태의 버튼은 텍스트 버튼이 아니라 아이콘 형태로 축소 노출한다.

### 3. 질문 리스트

#### 최근 7일 질문

- 오늘 포함 최근 7일 질문은 전체 펼침 상태로 노출한다.
- 예시: 오늘이 화요일이면 지난주 수요일 질문까지 펼침 상태로 노출한다.
- 구성 요소는 아래와 같다.
  - 날짜
  - 답변 개수
  - 질문
  - 글쓰기 또는 답변보기 아이콘
- 어제부터 7일 이내 질문은 답변이 20개씩 펼쳐진 상태로 노출한다.
- 답변이 펼쳐져 있는 질문만 스크롤 시 플로팅 상단 고정 대상이 된다.
- 접혀 있는 질문은 스크롤 시 상단 고정되지 않고 그대로 올라간다.

#### 8일 이전 질문

- 8일 이전 과거 질문은 답변이 닫힌 상태로 질문 리스트만 나열한다.
- 펼쳐진 질문과 닫힌 질문의 UI는 서로 다르게 구성한다.
- 닫힌 상태에서도 바로 작성 가능한 `작성하기` 버튼을 제공한다.
- 작성 후에는 `작성하기` 버튼이 `답변 보기` 아이콘으로 변경된다.
- `답변 보기` 선택 시 답변 상세 페이지를 호출한다.

### 4. 데일리 답변 리스트

- 답변은 기본값으로 최대 20개까지 노출한다.
- 하단에 `더보기 (n/300)` 버튼을 제공하여 답변을 20개씩 추가 노출한다.
- 모든 답변을 펼친 후에는 `전체 닫기` 기능을 제공한다.

#### 답변 구성 항목

- 프로필
- 닉네임
- 작성시간
- 답변 본문
- 좋아요 수
- 댓글 수

#### 상세 규칙

- 프로필에는 팔로우 중인 사용자일 경우 `v` 표시를 노출한다.
- 답변 본문은 최대 3줄까지 노출한다.
- 3줄을 초과하는 경우 본문을 아래로 더 펼치지 않고 답변 상세 페이지로 이동해 확인하도록 한다.
- 프로필 클릭 시 해당 사용자 프로필 페이지를 연다.
- 답변 영역 또는 댓글 아이콘 클릭 시 답변 상세 페이지를 호출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 모바일 | M-FD-010 |
| 웹 | W-FD-010 |

#### 작성 시간 표기 규칙

- `방금 전`: 0~59초
- `N분 전`: 1~59분
- `N시간 전`: 24시간 전까지
- `1월 25일`: 올해 작성분
- `2025년 1월 19일`: 이전 연도 작성분

#### 내 답변

- 나의 답변은 항상 질문 바로 아래에 상단 고정한다.
- 수정 기능은 답변 상세 페이지의 수정 화면으로 이동해 제공한다.
- 삭제 기능 선택 시 `답변을 삭제하시겠습니까? [취소/확인]` 확인 절차를 노출한다.
- `전체공개/나만보기` 전환은 별도 팝업 알럿 없이 즉시 동작한다.

### 5. 오늘의 콘텐츠

- 오늘의 질문과 연관된 `오늘의 콘텐츠`를 노출한다.
- 노출 위치는 답변 리스트 내 첫 번째 답변 아래이며, 추천 콘텐츠 1개씩 노출한다.
- 구성 항목은 아래와 같다.
  - 썸네일
  - 콘텐츠 소비시간
  - 콘텐츠명
  - 콘텐츠 소개글
  - 강사명 + 직함
  - 서브테마 유형
  - 조회수
- 콘텐츠 소비시간은 영상의 경우 `00:00`, 에세이의 경우 `00분` 형식으로 표시한다.
- 콘텐츠 소개글은 PC에서 최대 2줄까지 노출하고, 초과 시 말줄임 처리한다.
- 서브테마 유형 예시는 `대담`, `북러닝`, `에세이`이다.

### 6. 위즈덤스프링 답변

- 위즈덤스프링 답변은 오늘의 질문에 따라 매일 1개씩 등록된다.
- 질문과 동일한 API를 사용한다.
- 위즈덤스프링 답변은 답변 리스트의 첫 번째 답변으로 등록된다.
- 별도 고정 답변으로 처리하지는 않는다.
- 위즈덤스프링 답변에는 좋아요와 댓글 기능을 제공하지 않는다.

## 비고

- 답변 리스트 `더보기 (n/300)`의 최대 300이 고정 값인지, 전체 답변 수 예시인지 추가 확인이 필요하다.
- `팔로우중인 사용자 v표시`의 실제 아이콘/배지 디자인은 별도 정의가 필요하다.
