# HM-P022 전체 회차 팝업/바텀시트

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면 ID | HM-P022 |
| 모바일 화면 ID | M-HM-B022 |
| 웹 화면 ID | W-HM-P022 |
| 화면명 | 전체 회차 |
| 화면 유형 | 모바일 바텀시트 / 웹 팝업 |

## 화면 개요

현재 테마의 전체 콘텐츠를 한 번에 탐색할 수 있는 오버레이 화면이다. 모바일에서는 바텀시트, 웹에서는 모달 팝업으로 노출되며, 사용자는 가로 방향 스와이프 가능한 리스트에서 원하는 콘텐츠를 선택해 바로 학습화면으로 이동할 수 있다.

## 상세 설명

### 1. 진입 방식

- `전체 콘텐츠를 볼 수 있는 그리드/목록 아이콘` 클릭 시 오버레이를 노출한다.
- 디바이스별 노출 방식은 아래와 같다.

| 디바이스 | 노출 방식 |
| --- | --- |
| 모바일 | 바텀시트 |
| 웹 | 모달 팝업 |

### 2. 전체 콘텐츠 리스트 탐색

- 오버레이 내에서 전체 콘텐츠 리스트를 노출한다.
- 콘텐츠 리스트는 가로 방향으로 스와이프 가능해야 한다.
- 각 콘텐츠 카드는 아래 항목으로 구성한다.
  - 썸네일
  - 제목
  - 재생 중 프로그레스바
- 완료 표시 여부는 추가 검토 대상이다.

### 3. 재생 중 상태 표시

- 현재 시청 중인 콘텐츠 카드에는 `재생중` 뱃지를 노출한다.
- 재생 중 상태는 아래 방식 중 하나로 시각적으로 강조한다.
  - 썸네일 카드 테두리 하이라이트
  - 재생 아이콘 표시

### 4. 콘텐츠 선택

- 콘텐츠 클릭 시 해당 학습화면으로 이동한다.
- 이동과 동시에 현재 오버레이는 닫힌다.

### 5. 닫기

- `X` 버튼 클릭 시 오버레이를 닫고 기존 페이지로 복귀한다.
- 시트 또는 팝업 외부 영역 터치/클릭 시에도 오버레이를 닫고 기존 페이지로 복귀한다.

## 비고

- 완료 상태 표시 여부는 추가 정의가 필요하다.
- 가로 스와이프 방식이 카드 캐러셀인지, 리스트 단위 페이지 이동인지 구체 정의가 필요하다.
