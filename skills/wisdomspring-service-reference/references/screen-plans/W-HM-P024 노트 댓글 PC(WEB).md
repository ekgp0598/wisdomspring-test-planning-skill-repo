# W-HM-P024 노트 댓글 PC(WEB)

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 화면 ID | W-HM-P024 |
| 화면명 | 노트 댓글 PC(WEB) |
| 화면 유형 | 팝업 |

## 화면 개요

웹 학습화면에서 노트(댓글) 항목 선택 시 노출되는 댓글 상세 팝업이다. 선택한 댓글의 전체 내용과 대댓글 리스트를 확인하고, 즉시 대댓글을 작성할 수 있도록 구성한다.

## 상세 설명

### 1. 팝업 진입

- 좋아요 아이콘을 제외한 댓글 영역 전체 클릭 시 댓글 상세 팝업을 노출한다.
- 팝업 오픈 시 백그라운드는 딤 처리한다.
- 팝업 진입 시 대댓글 입력 필드에 포커스한다.

### 2. 댓글 상세

- 선택한 댓글의 전체 내용을 노출한다.
- 댓글 전체 펼침 상태로 표시한다.

### 3. 대댓글

- 선택한 댓글 하위의 대댓글 리스트를 노출한다.
- 입력 필드를 통해 대댓글을 작성할 수 있다.
- 입력 필드 플레이스홀더 또는 기능명은 `대댓글 작성`이다.
- 줄바꿈은 허용하지 않는다.
- 입력 필드는 최대 3줄까지 표시한다.
- 3줄을 초과하면 스크롤 처리한다.

### 4. 빈 상태

- 댓글이 없을 경우 빈 상태 메시지를 노출한다.
- 구체 문구는 추가 정의가 필요하다.

## 비고

- 본문에는 `댓글 페이지 팝업`과 `노트 댓글` 용어가 혼용되어 있어 실제 명칭 기준 확인이 필요하다.
- 댓글 없음 상태 문구는 별도 정의가 필요하다.
