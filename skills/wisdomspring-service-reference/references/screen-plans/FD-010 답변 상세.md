# FD-010 답변 상세

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면 ID | FD-010 |
| 모바일 화면 ID | M-FD-010 |
| 웹 화면 ID | W-FD-010 |
| 화면명 | 답변 상세 |
| 모바일 형태 | 페이지 이동 |
| 웹 형태 | 모달 팝업 |

## 화면 개요

질문 피드 내 특정 답변을 상세하게 확인하고, 좋아요, 댓글, 공유, 대댓글 작성까지 수행할 수 있는 화면이다. 모바일은 페이지 이동 방식, 웹은 모달 팝업 방식으로 노출된다.

## 상세 설명

### 1. 진입 경로

- 답변 클릭 시 진입한다.
- 댓글 버튼 클릭 시 진입한다.
- 질문 카드의 `답변보기 (v 아이콘)` 클릭 시 진입한다.

### 2. 답변 내용

- 답변 내용 영역은 상단 고정하지 않는다.
- 답변 내용 영역 내에서 스크롤한다.

#### 구성 항목

- 프로필 이미지
- 닉네임
- 본문 내용

#### 프로필 영역

- 팔로우 중인 경우
  - 프로필 이미지에 `v` 표시를 노출한다.
  - 우측에 `팔로잉` 텍스트를 노출한다.
- 팔로우하고 있지 않은 경우
  - 우측에 `팔로우` 텍스트를 노출한다.
  - `팔로우` 클릭 시 `팔로잉`으로 텍스트를 변경한다.
- 본인 답변일 경우
  - `나` 표시를 노출한다.
  - 더보기 버튼 드롭다운을 제공한다.
  - 드롭다운 항목은 아래와 같다.
    - 수정
    - 삭제
    - 전체공개/나만보기

#### 본문

- 본문은 답변 상세 영역 내에서 확인한다.

#### 액션

- 좋아요
  - 클릭 시 아이콘을 filled 상태로 변경한다.
  - 카운트를 증가시킨다.
- 댓글
  - 대댓글 영역 및 작성 영역과 연계된다.
- 공유
  - 웹은 공유 팝업을 노출한다.
  - 모바일은 공유 바텀바를 노출한다.

### 3. 대댓글

- 대댓글 리스트를 노출한다.
- 각 대댓글의 구성 항목은 아래와 같다.
  - 프로필 이미지
  - 닉네임
  - 댓글
- 댓글은 미리보기 기준 최대 3줄까지 노출한다.
- 3줄을 초과할 경우 `더보기` 처리한다.

### 4. 대댓글 작성

- 대댓글 작성 입력 필드를 제공한다.
- 줄바꿈은 허용하지 않는다.
- 입력 필드는 최대 3줄까지 노출한다.
- 3줄을 초과하면 스크롤 처리한다.

## 비고

- `답변보기 (v 아이콘)`의 실제 아이콘 형태는 추가 정의가 필요하다.
- 사용자 입력 원문에 `전체공기/나만보기`로 기재되어 있으나, 문맥상 `전체공개/나만보기` 의미로 해석해 반영했다. 필요 시 표기 기준 확인이 필요하다.
