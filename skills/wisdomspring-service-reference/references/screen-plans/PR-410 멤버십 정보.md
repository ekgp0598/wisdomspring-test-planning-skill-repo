# PR-410 멤버십 정보

| 화면명 | 멤버십 정보 |

## 화면 개요

사용자의 멤버십 가입 상태와 결제 상태를 확인하는 화면이다. 구독중, 구독중(해지예약), 사용정지 상태에 따라 타이틀과 정보 구성이 달라지며, 멤버십 유형/상태/CTA를 동일한 구조로 그룹핑해 제공한다.

## 공통 변경 사항

- 타이틀은 상태별 텍스트로 변경한다.
- 멤버십 종류를 표시한다.
- 멤버십 종류는 `월간 멤버쉽` 또는 `연간 멤버쉽`으로 표시한다.
- 구독 상태를 표시한다.
- 구독 상태 값은 아래 중 하나를 사용한다.
  - `구독중`
  - `구독중(해지예약)`
  - `사용정지`
- 구독기간과 구독료 위치를 상단으로 이동한다.
- 구독정보 영역은 `멤버십 유형/상태 + CTA`를 그룹핑하는 방식으로 통일한다.
- 기존 `결제 수단`, `결제 수단 관리` 항목은 삭제한다.
- 대신 `결제관리` 링크를 제공하고, 클릭 시 `결제관리` 페이지로 이동하도록 한다.

## 상태별 상세 설명

### 1. 구독중

- 타이틀에 구독중 상태에 맞는 텍스트를 노출한다.
- 멤버십 종류를 `월간 멤버쉽` 또는 `연간 멤버쉽`으로 표시한다.
- 구독 상태는 `구독중`으로 표시한다.
- 구독정보 영역에는 아래 항목을 제공한다.
  - 멤버십 종류
  - 해지신청 버튼
  - 다음 결제일 정보

### 2. 구독중(해지예약)

- 타이틀에 해지예약 상태에 맞는 텍스트를 노출한다.
- 멤버십 종류를 `월간 멤버쉽` 또는 `연간 멤버쉽`으로 표시한다.
- 구독 상태는 `구독중(해지예약)`으로 표시한다.
- 구독정보 영역에서 기존 `해지 예약일` 워딩은 `해지 예정일`로 변경한다.

### 3. 사용정지

- 타이틀에 사용정지 상태에 맞는 텍스트를 노출한다.
- 멤버십 종류를 `월간 멤버쉽` 또는 `연간 멤버쉽`으로 표시한다.
- 구독 상태는 `사용정지`로 표시한다.
- 구독정보 영역에는 아래 항목을 제공한다.
  - 결제실패 N회
  - 멤버십 즉시 연장하기
  - 다음 결제일
- `멤버십 즉시 연장하기` 클릭 시 멤버십 가입 페이지로 이동한다.

#### 결제실패 회차별 처리

- 결제실패 1회, 2회까지는 `구독중` 카드 + 결제실패 구독정보 조합으로 노출한다.
- 결제실패 3회 이상부터는 `사용정지` 카드 + 결제실패 구독정보 조합으로 노출한다.
- 그 외 구성은 동일하게 유지한다.

## 변경 없음

- 미가입자이면서 구독이력 존재 상태

위 두 케이스는 기존과 동일하며, 별도 디스크립션 변경 사항은 없다.
