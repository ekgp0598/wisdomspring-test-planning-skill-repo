## 화면 개요

프로필 화면에서 사용자가 작성한 댓글 이력을 확인하는 화면이다. 노트 댓글과 답변 댓글을 모두 포함하며, 내 댓글 정보와 원문 정보를 함께 노출하고 해당 원문 상세 화면으로 이동할 수 있도록 구성한다.

## 상세 설명

### 1. 댓글 대상

- 노트 댓글과 답변 댓글을 모두 표시한다.

### 2. 댓글 리스트 구성

- 각 댓글 항목에는 아래 정보를 노출한다.
  - 나의 프로필
  - 댓글 작성일
  - 댓글 내용
  - 원문 작성자 프로필
  - 원문 내용

#### 노출 규칙

- 댓글 작성일은 `YYYY.MM.DD` 형식으로 표시한다.
- 댓글 내용은 최대 2줄까지 표시한다.
- 2줄을 초과하면 말줄임 처리한다.
- 원문 내용도 최대 2줄까지 표시한다.
- 2줄을 초과하면 말줄임 처리한다.

### 3. 클릭 동작

- 나의 댓글 영역 클릭 시 원문 유형에 따라 상세 화면으로 이동 또는 팝업/바텀시트를 호출한다.
- 원문 영역 클릭 시에도 동일한 규칙을 적용한다.

## 비고

- `페이지 이동` 표현은 답변 상세에 명확하나, 노트 상세는 웹 팝업/모바일 바텀시트 호출 방식으로 구분되어 있다.
- `나의 프로필`, `원문 작성자 프로필`에 어떤 세부 정보까지 포함되는지는 추가 정의가 필요하다.
