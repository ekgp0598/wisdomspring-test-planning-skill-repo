# HM-020 학습화면-영상

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면 ID | HM-020 |
| 모바일 화면 ID | M-HM-020 |
| 웹 화면 ID | W-HM-020 |
| 화면명 | 학습화면-영상 |
| 화면 유형 | 페이지 |

## 화면 개요

사용자가 영상 콘텐츠를 클릭하여 진입했을 때 노출되는 학습화면이다. 플레이어를 중심으로 콘텐츠 정보, 이전/다음 화 이동, 댓글, 추천 콘텐츠를 제공하며, 디바이스별로 댓글 노출 방식에 차이가 있다.

## 상세 설명

### 1. 플레이어

- 콘텐츠 영역 진입 시 영상을 자동 재생한다.
- 테마 상세페이지로 이동하는 뒤로가기 버튼을 제공한다.
- 단, 브라우저 또는 앱의 페이지 뒤로가기 동작일 경우에는 이전 페이지로 이동한다.
- `전체 회차 보기` 아이콘 클릭 시 회차 목록을 노출한다.

| 디바이스 | 호출 화면 | 방식 |
| --- | --- | --- |
| 모바일 | M-HM-B022 | 바텀시트 |
| 웹 | W-HM-P022 | 팝업 |

- 재생시간은 `현재 재생 시간 / 총 콘텐츠 시간` 형식으로 표시한다.

### 2. 콘텐츠 정보

#### 기본 노출

- 서브테마명
- 콘텐츠명

#### 펼침 시 추가 노출

- 콘텐츠 소개글
- 태그
- 강사 사진
- 강사명

#### 인터랙션

- 좋아요
  - 클릭 시 카운트가 증가한다.
  - 아이콘은 filled 상태로 변경한다.
- 북마크
  - 아이콘은 filled 상태로 표시한다.
  - 클릭 시 나의 북마크에 추가한다.
- 공유
  - 바텀시트 형태로 펼친다.
  - 제공 항목은 `카카오톡`, `Facebook`, `링크 복사`이다.
  - 각 항목은 아이콘 + 텍스트 조합으로 노출한다.
  - `링크 복사` 클릭 시 하단 토스트 팝업 `링크가 복사되었습니다.`를 노출한다.
  - `카카오톡`, `Facebook` 클릭 시 별도 복사 토스트 없이 각 앱 또는 브라우저 공유로 전환한다.
  - 공유 앱 호출 실패 시 하단 토스트 팝업 `공유를 실행할 수 없습니다.`를 노출한다.

### 3. 이전/다음 화 보기

- 이전 화, 다음 화 이동 영역을 제공한다.
- 노출 항목은 아래와 같다.
  - 이전화 또는 다음화 구분
  - 서브테마 유형
  - 썸네일
  - 콘텐츠 제목
- 콘텐츠 제목은 최대 2줄까지 노출하며, 초과 시 말줄임 처리한다.

#### 상태 케이스

- 이전 화와 다음 화가 모두 있는 경우
  - 두 항목을 모두 노출한다.
- 이전 화가 없는 경우
  - 아이콘과 함께 `이전 콘텐츠가 없습니다.` 문구를 노출한다.
- 다음 화가 없는 경우
  - 아이콘과 함께 `마지막 콘텐츠입니다.` 문구를 노출한다.

### 4. 노트

- 웹은 학습화면 내에 노트를 직접 노출한다.
- 모바일은 노트를 바텀시트로 노출한다.

| 디바이스 | 호출 화면 | 방식 |
| --- | --- | --- |
| 모바일 | M-HM-B023 | 바텀시트 |
| 웹 | W-HM-020 내 영역 노출 | 인라인 |

#### 노트 리스트

- 최신순 기준 상단부터 노출한다.
- 한 번에 최대 10개까지 노출한다.
- 추가 노트는 페이지네이션으로 처리한다.
- 총 노트 개수를 함께 표시한다.
- 각 노트의 구성 항목은 아래와 같다.
  - 프로필 사진
  - 닉네임
  - 노트 본문
  - 작성 시각
  - 좋아요
  - 댓글 수
- 노트 본문은 최대 3줄까지 노출하고, 초과 시 말줄임 처리한다.
- 좋아요 아이콘 클릭 시 카운트가 증가한다.
- 좋아요 아이콘은 filled 상태로 변경한다.
- 댓글 아이콘 클릭 시 해당 노트의 댓글 팝업을 노출한다.

| 디바이스 | 호출 화면 |
| --- | --- |
| 웹 | W-HM-P024 |

#### 노트 입력

- 노트 입력 필드를 제공한다.
- 줄바꿈은 허용하지 않는다.
- 입력 필드 높이는 최대 3줄까지 늘어난다.
- `ENTER` 입력 시 노트를 등록한다.

#### 노트 없음 상태

- 노트가 없을 경우 아래 문구를 노출한다.
- `등록된 노트가 없습니다. 첫 번째 노트를 남겨보세요.`

### 5. 추천 콘텐츠

- 현재 재생 중인 콘텐츠와 연관된 로직으로 추천 콘텐츠를 노출한다.
- 추천은 현재 테마와 무관하게 구성될 수 있다.
- 추천 대상은 `TALK`, `BOOK`, `ESSAY`를 모두 포함한다.
- 구성 항목은 아래와 같다.
  - 테마명
  - 서브테마 유형
  - 썸네일
  - 콘텐츠명

## 비고

- 추천 콘텐츠의 상세 추천 로직은 별도 정의가 필요하다.
