# PR-140 프로필-팔로워

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 화면 ID | PR-140 |
| 화면명 | 프로필 > 팔로워 |

## 화면 개요

프로필 화면에서 팔로워 및 팔로잉 관계를 확인하고 관리하는 화면이다. 기존 탭명 `팔로우`를 `팔로워`로 변경하며, 사용자는 팔로워/팔로잉 의미를 툴팁으로 확인하고 리스트에서 즉시 팔로우/언팔로우 액션을 수행할 수 있다.

## 상세 설명

### 1. 탭 명칭 변경

- 기존 `팔로우` 탭명을 `팔로워`로 수정한다.

### 2. 용어 정의

- `팔로워`
  - 나를 팔로잉하는 사람
- `팔로잉`
  - 내가 팔로잉하는 사람
- `팔로우`
  - 팔로잉하기 액션

### 3. 팔로워 수 안내

- 조회 중인 프로필 사용자를 팔로우하고 있는 `팔로워` 수를 표시한다.
- PC에서는 호버 시 툴팁을 노출한다.
- 모바일에서는 클릭 시 툴팁을 노출한다.

### 4. 팔로워/팔로잉 리스트

- 팔로워/팔로우 리스트를 노출한다.
- 각 항목의 구성 요소는 아래와 같다.
  - 프로필
  - 닉네임
  - 팔로우 여부 버튼

#### 상태별 노출 규칙

- 내가 해당 사용자를 팔로잉하고 있는 경우
  - 프로필에 `v` 아이콘을 노출한다.
  - 버튼은 `팔로잉`으로 표시한다.
  - `팔로잉` 클릭 시 팔로잉 해제 처리한다.
  - 해제 후 프로필의 `v` 아이콘은 제거한다.
  - 버튼은 `팔로우`로 변경한다.
- 내가 해당 사용자를 팔로잉하고 있지 않은 경우
  - 버튼은 `팔로우`로 표시한다.
  - `팔로우` 클릭 시 팔로잉 처리한다.
  - 처리 후 프로필에 `v` 아이콘을 노출한다.
  - 버튼은 `팔로잉`으로 변경한다.

## 비고

- `팔로워/팔로우 리스트` 표현상 실제 화면이 `팔로워`와 `팔로잉` 두 탭인지, 단일 리스트인지 추가 확인이 필요하다.
- 툴팁에 실제로 어떤 문구를 노출할지는 별도 정의가 필요하다.
