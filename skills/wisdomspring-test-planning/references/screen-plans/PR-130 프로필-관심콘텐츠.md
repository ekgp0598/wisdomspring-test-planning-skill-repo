# PR-130 프로필-관심콘텐츠

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 화면 ID | PR-130 |
| 화면명 | 프로필 > 관심콘텐츠 |

## 화면 개요

사용자가 북마크한 콘텐츠를 모아 확인하는 프로필 탭 화면이다. 콘텐츠 썸네일, 제목, 북마크 상태, 북마크 일자를 확인할 수 있으며, 북마크 해제는 즉시 시각적으로 반영하되 실제 리스트 제거는 화면 재진입 시 적용된다.

## 상세 설명

### 1. 콘텐츠 리스트

- 콘텐츠 리스트는 무한 스크롤 방식으로 동작한다.
- PC와 모바일 모두 15개 단위로 로드한다.
- 각 콘텐츠 항목의 구성 요소는 아래와 같다.
  - 콘텐츠 썸네일
  - 제목
  - 북마크 아이콘
  - 북마크한 날짜

### 2. 북마크 기능

- 북마크 아이콘 클릭 시 filled 아이콘을 line 아이콘으로 변경한다.
- 하단 토스트 팝업 `관심 콘텐츠에서 삭제되었습니다.`를 노출한다.
- 토스트 내 `되돌리기` 클릭 시 북마크 아이콘을 다시 filled 상태로 변경한다.

### 3. 북마크 해제 후 리스트 반영 규칙

- 북마크를 해제해도 현재 화면에서는 콘텐츠가 즉시 사라지지 않는다.
- 콘텐츠는 현재 화면에 그대로 남아 있다.
- 화면 이탈 후 재진입 시 리스트에서 제거한다.
- 재진입 예시는 아래와 같다.
  - 탭 이동 후 복귀
  - 새로고침

### 4. 콘텐츠 진입

- 콘텐츠 클릭 시 콘텐츠 유형에 따라 학습화면으로 직접 이동한다.

| 콘텐츠 유형 | 이동 화면 |
| --- | --- |
| 영상 | HM-020 |
| 아티클 | HM-021 |

- 삭제되었거나 비노출 처리된 콘텐츠는 상세 진입 대신 토스트 `현재 볼 수 없는 콘텐츠입니다.`를 노출한다.

## 비고

- 북마크한 날짜의 표기 형식은 추가 정의가 필요하다.
