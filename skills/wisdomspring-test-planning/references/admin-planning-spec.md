# WisdomSpring Admin Planning Spec

이 문서는 WisdomSpring admin 기능의 상세 기획 기준본이다. 빠른 정책 판단은 `admin-policy-summary.md`를 먼저 보고, 목록 컬럼, 상세 필드, 등록 제약, 수정/삭제 제한, 운영 플로우까지 확인해야 할 때는 이 문서를 사용한다.

## 개요

- 대상 영역
  - 테마 관리
  - 오늘의 질문 관리
  - 오늘의 콘텐츠 관리
  - 저널 관리 > 답변 기록 관리
  - 콘텐츠 노트 관리
- 목적
  - admin 정책과 front 동작의 정합성을 검증한다.
  - admin 목록, 등록, 수정, 삭제, 노출 제약과 조회 기준을 테스트 근거로 사용한다.
- 사용 원칙
  - 일정 기반 노출, 중복 제한, 수정/삭제 잠금, hidden/delete 구분은 버그 판단의 핵심 근거로 본다.
  - 수치가 확정되지 않은 `N bytes` 항목은 추정하지 않고 TBD로 유지한다.

## 변경 Flow

### 등록 Flow

- 기존: 테마/서브테마 등록 -> 콘텐츠 등록 시 테마/서브테마 지정
- 변경: 콘텐츠 등록 -> 테마/서브테마 등록 시 콘텐츠 불러오기하여 등록

### 질문 관리 Flow

- 기존: 데일리 메세지 등록 -> 편성표에서 날짜별로 불러와 지정
- 변경: 데일리 메세지 등록 시 날짜 지정

## 주요 정책

### 1. 테마 관리

#### 1.1 테마 노출 정책

- 테마는 오픈 일자 기준으로 프론트에 노출된다.
- 다음 테마가 오픈되기 전까지 현재 테마는 `NEW` 상태를 유지한다.
- 어드민 목록 검색/조회는 등록일이 아닌 오픈 일자 기준으로 한다.

#### 1.2 테마 썸네일 노출 정책

- 썸네일은 최소 1개, 최대 10개 등록할 수 있다.
- 각 썸네일에는 노출 날짜를 지정할 수 있다.
- 오늘 기준 가장 최신 날짜로 지정된 썸네일이 프론트에 노출된다.

#### 1.3 서브테마 유형별 콘텐츠 등록 제약

- `TALK`, `BOOK`, `ETC-영상`: 동영상 콘텐츠만 등록 가능
- `ESSAY`, `ETC-아티클`: 아티클 콘텐츠만 등록 가능

#### 1.4 서브테마 오픈 상태 정책

- 오픈 일정 도래 전: `COMING SOON`
- 오픈 후 1주일간: `NEW`
- 오픈 이후라도 오픈 일정을 수정하면 상태가 변경된 날짜 기준으로 재적용된다.

#### 1.5 테마 노출 ON/OFF 제약

- 서브테마 내 콘텐츠가 하나도 등록되지 않은 경우 노출 `ON` 처리 불가

#### 1.6 콘텐츠 중복 등록 제한

- 이미 다른 테마/서브테마에 등록된 콘텐츠는 중복 등록 불가

### 2. 오늘의 질문 관리

#### 2.1 질문 등록 제약

- 날짜 중복 등록 불가
- 이미 질문이 등록된 날짜는 선택할 수 없다.
- 과거 날짜 등록 불가
- 오늘 이전 날짜로 질문을 등록할 수 없다.

#### 2.2 질문 수정/삭제 제약

- 이미 오픈된 과거 질문은 수정 및 삭제 불가

### 3. 오늘의 콘텐츠 관리

#### 3.1 콘텐츠·문장 통합 정책

- 오늘의 문장(발췌문)은 항상 오늘의 콘텐츠와 함께 연결되어 관리된다.
- 기존 `콘텐츠 질문` 기능은 삭제하고 오늘의 콘텐츠 관리로 통합한다.

#### 3.2 콘텐츠 등록 제약

- 날짜 중복 등록 불가
- 이미 콘텐츠가 등록된 날짜는 선택할 수 없다.
- 과거 날짜 등록 가능

#### 3.3 콘텐츠·문장 수정/삭제 제약

- 이미 오픈된 날짜의 콘텐츠 및 문장은 수정 및 삭제 불가

### 4. 답변 기록 관리

#### 4.1 노출 관리 정책

- 답변을 미노출 처리하더라도 어드민 관리 목록에서는 계속 표시된다.

#### 4.2 답변 삭제 정책

- 삭제 처리된 답변은 어드민 관리 목록에서 미노출 처리된다.

### 5. 콘텐츠 노트 관리

#### 5.1 노트 삭제 정책

- 노트 삭제는 실제 데이터 삭제가 아닌 미노출 처리다.
- 삭제(미노출) 처리된 노트는 어드민 노트 목록에서 표시되지 않는다.

## 기능 정책

### 1. 테마 관리

#### 1.1 테마 목록

검색/조회 기준: 등록일이 아닌 오픈 일자 기준으로 검색/조회

| 컬럼명 | 설명 |
| --- | --- |
| No | 순번 |
| 오픈일 | 테마 오픈 일자 |
| 테마명 | 테마 이름 |
| 카테고리 | 테마 카테고리 |
| 서브테마 수 | 등록된 서브테마 개수 |
| 콘텐츠 수 | 등록된 콘텐츠 개수 |
| 등록일 | 테마 등록 일자 |
| 등록자 ID | 등록한 관리자 ID |

- `테마 등록` 버튼 클릭 시 테마 등록 페이지로 이동

#### 1.2 테마 기본 정보

| 항목 | 설명 |
| --- | --- |
| 테마명 | 테마 이름 |
| 카테고리 | 테마 카테고리 선택 |
| 오픈 일자 | 프론트 노출 기준 일자 |
| 테마 소개 | 썸네일 및 소개문구 |
| 노출 ON/OFF | 프론트 노출 여부 토글 |

오픈 일자 정책

- 오픈 일자에 따라 프론트에 노출됨
- 다음 테마가 오픈될 때까지 `NEW` 뱃지 표시
- 어드민 리스트는 오픈 일자 기준으로 검색/조회 가능

테마 소개 정책

- 썸네일 및 소개문구를 최소 1개, 최대 10개 등록 가능
- 각 썸네일마다 노출 날짜 지정 가능
- 오늘 기준 가장 최신 날짜로 지정된 썸네일이 노출됨
- 썸네일 소개문구: 최대 20자 제한
- `<br>` 태그 줄바꿈 지원

#### 1.3 서브테마 관리

| 항목 | 설명 |
| --- | --- |
| 서브테마 유형 | 유형 선택 (`TALK` / `BOOK` / `ESSAY` / `ETC-아티클` / `ETC-영상`) |
| 서브테마 유형명 | 유형 표시명 (최대 8bytes: 한글 4자, 영문 8자) |
| 서브테마명 | 서브테마 이름 (`N bytes` 제한) |
| 서브테마 소개 | 서브테마 설명 (`N bytes` 제한) |
| 썸네일 | 서브테마 대표 이미지 |
| 콘텐츠 | 서브테마에 등록할 콘텐츠 |

서브테마 추가/삭제 정책

- 서브테마는 최소 1개, 최대 10개 등록 가능
- 서브테마 순서 변경: 드래그 앤 드롭 지원
- 모든 설정값 수정 가능

기본 세팅

| 순번 | 유형 | 서브테마명 (Prefilled) |
| --- | --- | --- |
| 1 | TALK | 대담 |
| 2 | BOOK | 북러닝1 |
| 3 | BOOK | 북러닝2 |
| 4 | ESSAY | 에세이 |

- 추가하는 서브테마는 서브테마명이 빈 값으로 제공됨

서브테마 유형별 콘텐츠 등록 규칙

| 유형 | 등록 가능 콘텐츠 유형 |
| --- | --- |
| TALK | 동영상 콘텐츠만 등록 가능 |
| BOOK | 동영상 콘텐츠만 등록 가능 |
| ETC-영상 | 동영상 콘텐츠만 등록 가능 |
| ESSAY | 아티클 유형만 등록 가능 |
| ETC-아티클 | 아티클 유형만 등록 가능 |

서브테마 오픈 일정 정책

- 오픈 전: `COMING SOON`
- 오픈 후 1주일: `NEW`
- 오픈 이후 날짜 수정 시 변경된 날짜 기준으로 재적용

콘텐츠 등록 정책

- 콘텐츠가 등록되지 않아도 테마 저장 가능
- 단, 콘텐츠가 없을 경우 노출 `ON` 처리 불가

등록된 콘텐츠 리스트 노출 항목

| 항목 | 설명 |
| --- | --- |
| 썸네일 | 콘텐츠 대표 이미지 |
| 제목 | 콘텐츠 제목 |
| 강사(저자) | 강사 또는 저자명 |
| 분수 | 콘텐츠 재생 시간 |
| 태그 | 콘텐츠 태그 |
| 상세보기 버튼 | 클릭 시 콘텐츠 상세 페이지로 이동 |
| 삭제 버튼 | 해당 콘텐츠 삭제 |

#### 1.4 서브테마 내 콘텐츠 등록

- 모달 팝업을 통해 콘텐츠 불러오기
- 콘텐츠 등록/삭제 및 순서 변경(드래그 앤 드롭) 가능
- 모달 진입 시 별도 검색 없이 최신 등록순 콘텐츠 리스트 자동 표시

불러오기 모달 내 콘텐츠 리스트 노출 항목

| 항목 |
| --- |
| 썸네일 |
| 제목 |
| 강사(저자) |
| 분수 |
| 등록 날짜 |
| 태그 |

콘텐츠 등록 제약

- 이미 다른 테마/서브테마에 등록된 콘텐츠는 등록 불가

### 2. 오늘의 질문 관리

- 기존 `콘텐츠 질문`의 콘텐츠 등록 기능은 오늘의 콘텐츠 관리로 통합하여 삭제

#### 2.1 목록 조회

캘린더형 뷰

- 편성표 형태 UI
- 각 날짜 셀에 질문 텍스트 직접 표시
- 셀 구성
  - 좌상단: 날짜 숫자
  - 본문: `오늘의 질문` 라벨 + 질문 내용(최대 2줄, 말줄임)
- 질문 미등록 날짜: 본문에 `+ 등록` 플레이스홀더 표시 -> 클릭 시 신규 작성
- 날짜 자동 입력 가능, 수정 가능

리스트형 뷰 검색 조건

| 항목 | 설명 |
| --- | --- |
| 기간 | 오픈 일자 기준 |
| 키워드 검색 대상 | 질문 / 아이디 (기본값: 질문) |

리스트형 뷰 테이블 항목

| 컬럼명 | 설명 |
| --- | --- |
| No | 순번 |
| 날짜 | 오픈 일자 (`YYYY-MM-DD`) |
| 내용 | 질문 내용 |
| 답변 수 | 해당 질문의 답변 수 |
| 작성자 ID | 등록한 관리자 ID |

- `답변 수` 클릭 시 `저널 관리 > 답변 기록 관리`로 이동하며 해당 질문으로 필터링된 답변 목록을 보여준다.

#### 2.2 질문 등록

- 모달 팝업으로 등록

| 항목 | 규칙 |
| --- | --- |
| 오픈일 | 이미 질문이 등록된 날짜는 캘린더 피커에서 선택 불가 / 과거 날짜 등록 불가 |
| 질문 내용 | `N bytes` 제한 / 빈 값이면 등록 불가 |

#### 2.3 질문 조회 / 수정

조회 항목

| 항목 |
| --- |
| 오픈일 |
| 질문 내용 |
| 작성자 ID |
| 답변 수 |

수정/삭제 제약

- 이미 오픈된 과거 질문은 수정 및 삭제 불가

### 3. 오늘의 콘텐츠 관리

- 기존 `오늘의 콘텐츠`와 `오늘의 문장`을 통합 관리
- 오늘의 문장은 항상 오늘의 콘텐츠와 연결되어 함께 노출

#### 3.1 목록 조회

캘린더형 뷰

- 편성표 형태 UI
- 각 날짜 셀에 오늘의 콘텐츠와 오늘의 문장을 함께 표시
- 날짜 셀 클릭 시 오늘의 콘텐츠를 불러오고 하단에 오늘의 문장을 작성

리스트형 뷰 검색 조건

| 항목 | 설명 |
| --- | --- |
| 기간 | 오픈 일자 기준 |
| 키워드 검색 대상 | 콘텐츠명 / 오늘의 문장 / 아이디 (기본값: 콘텐츠명) |

리스트형 뷰 테이블 항목

| 컬럼명 | 설명 |
| --- | --- |
| No | 순번 |
| 오픈일 | 오픈 일자 (`YYYY-MM-DD`) |
| 콘텐츠명 | 콘텐츠 제목 |
| 오늘의 문장 | 발췌 문장 |
| 작성자 ID | 등록한 관리자 ID |

- 기존 `Type` 컬럼 삭제
- 날짜는 등록일자가 아닌 오픈 일자 기준으로 표시

#### 3.2 오늘의 콘텐츠 등록

- 모달 팝업으로 등록
- 항목 전체 필수값

| 항목 | 규칙 |
| --- | --- |
| 오픈일 | 이미 콘텐츠가 등록된 날짜는 캘린더 피커에서 선택 불가 / 과거 날짜 등록 가능 |
| 콘텐츠 선택 | 콘텐츠 검색 코드 `A-TM-011` 기준 / 콘텐츠 유형 무관 선택 가능 |
| 오늘의 문장 | `N bytes` 제한 |

#### 3.3 오늘의 콘텐츠 조회 / 수정

조회 항목

| 항목 |
| --- |
| 오픈일 |
| 콘텐츠 |
| 오늘의 문장 |
| 작성자 ID |

수정/삭제 제약

- 이미 오픈된 날짜의 콘텐츠 및 문장은 수정 및 삭제 불가
